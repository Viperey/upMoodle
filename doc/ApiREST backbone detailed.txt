==USER==

=GetUser=
	[GET] user/{id}
		200 - Returns user's model.
		400 - ErrorMessage(1) + redirects to login

=Get user by level=
	[GET] users/level/{id}
		200 - Returns user's model.
		400 - ErrorMessage(1) + redirects to login

=DeleteUser=
	[DELETE] user/remove/
		200 - Redirects to login.
		400 - ErrorMessage(1)

=ModifyUser=
	[PUT] user/update/
		200 - Returns user's model
		400 - ErrorMessage(1)
